# KKI2026 Monorepo Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Membuat `D:/KKI2/KKI2026` sebagai monorepo software-only yang menjalankan dashboard JavaScript dan bridge Python dari satu root.

**Architecture:** Pertahankan `dashboard/` dan `asv_dashboard_backend/` pada path yang sama relatif terhadap root baru agar import, Vite, deployment Raspberry Pi, dan test tidak berubah. NPM workspace hanya mendaftarkan `dashboard`; root scripts mengorkestrasi package JavaScript dan perintah Python tanpa menambah Turborepo.

**Tech Stack:** NPM workspaces, TanStack Start/Vite, Python 3 + FastAPI, pytest, Supabase migrations, Git.
